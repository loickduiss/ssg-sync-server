Operating modes
MODE 1 — DROK precision
48V PSU → Relay1 NC → DROK buck → motor · exact voltage control via encoder
MODE 2 — PWM speed
48V PSU → Relay2 NC → PWM controller → motor · speed knob control
MODE 3 — Feedback loop
Generator → banana port → Relay3 NO → DROK → motor · grid severed · isolated
MODE 4 — BASH signal
Uno D9 PWM → BASH pre-amp RCA → BASH amp → test load · Relay4 enables
Jumpers J1–J3
On relay output terminals — reconfigure routing without rewiring
Schottky diodes
One-way valves on grid + feedback paths — prevent backfeed through relay
Loop closure rule
Relay1 must NOT flip to NO until DROK output is within ±5% of motor rated V

Energy audit equations
Pin (grid watts)
ACS758#1 reading × ZMPT grid voltage
Pout (gen watts)
ACS758#2 reading × ZMPT generator voltage
Feedback watts
ACS758#3 reading × DROK output voltage
Efficiency η
Pout ÷ Pin × 100%
Deficit
Pin − Pout = wattage gap preventing self-sustainability

Scenario 1 — BASH signal generation MAPPED
Trigger
User selects BASH mode on TFT touchscreen
Signal chain
TFT → Uno D9 PWM → BASH pre-amp RCA → BASH amplifier → output
Physical controls
POT1 (Hz) + POT2 (Volume) on pre-amp board — Mega reads via ADC
Waveforms
Sine · Square · PWM · DC · Triangle · Sawtooth — user selects on TFT
OSC channels
Ch1: injected signal pre-amp · Ch5: amplifier output · ghost overlay for recall
Preset system
Save type+Hz+duty+amplitude+ghost waveform → recall loads params + ghost on OSC → turn POTs to match
Scenario 2 — Motor onboarding wizard MAPPED
Step 1
Motor type: brushed DC / BLDC / AC induction / stepper / unknown
Step 2
Enter label data: V, A, W, RPM, Kv — system calculates missing values
Step 3

Step 4
Select monitors → configures display layout + relay routing

Scenario 3 — Unknown motor identification MAPPED
Tests in order
Wire count → winding resistance → back-EMF spin test (OSC Ch3) → no-load current → RPM at test V → stall current (brief <1s)
Output
Auto-fills: motor type · winding R · Kv · no-load RPM · estimated power · stall current
Scenario 4 — Grid excitation → loop closure → decay analysis PENDING

Phase 2 — Monitor

Phase 3 — Isolate
